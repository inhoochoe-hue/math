# 학원용 중등 수학 난이도별 문항 생성기

HWP/HWPX/DOCX/TXT 문제 자료를 읽어 원본의 핵심 개념과 풀이 구조를 분석하고,
`하·중하·중·중상·상` 5단계로 세분화된 신규 문항을 생성합니다.

## 학원용 기능

- 웹 화면에서 문제 자료 업로드
- 난이도별 출제 문항 수 직접 설정
- 객관식·주관식·서술형 선택
- 원본 문제 자동 분리 및 단원·세부유형 분석
- 숫자만 바꾸지 않는 조건·질문·상황 변형
- 객관식 보기 5개 및 오답 설계
- SymPy 계산 검산
- 원본 및 생성 문항 사이 유사도·중복 검사
- 품질검사 실패 시 자동 재생성
- 학생용 문제지와 교사용 정답·해설 DOCX 분리
- SQLite 문제은행 누적 저장
- 전체 생성 데이터 JSON 보존

## 난이도 기준

| 난이도 | 출제 기준 |
|---|---|
| 하 | 핵심 개념 바로 적용, 계산 1~2단계 |
| 중하 | 간단한 식 변형 또는 조건 해석 1회 |
| 중 | 조건 2개 이상 연결, 표준 응용 상황 수식화 |
| 중상 | 역해석, 다단계 식 변형 또는 경우 구분 |
| 상 | 복합 조건, 숨은 제약, 역문제와 고난도 추론 |

## 가장 쉬운 실행 방법 — Windows

1. 압축을 풉니다.
2. 폴더 안의 `.env.example`을 복사해 `.env`로 이름을 바꿉니다.
3. `.env` 파일에 API 키를 입력합니다.

```env
OPENAI_API_KEY=sk-실제키
OPENAI_MODEL=gpt-5-mini
ACADEMY_NAME=OO수학학원
```

4. `실행_웹화면.bat`를 더블클릭합니다.
5. 브라우저가 열리면 HWP 파일을 올리고 난이도별 문항 수를 설정합니다.

> `.hwp`를 직접 읽으려면 Windows에 한컴오피스 한글이 설치되어 있어야 합니다.
> 변환 오류가 나면 한글에서 파일을 `.hwpx`로 다른 이름 저장한 뒤 HWPX를 올리면 더 안정적입니다.

## 터미널 실행

```bash
python -m venv .venv
```

Git Bash:

```bash
source .venv/Scripts/activate
```

PowerShell:

```powershell
.\.venv\Scripts\Activate.ps1
```

패키지 설치 및 웹 실행:

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 명령줄 실행

```bash
python main.py "input/역삼중.hwp" --distribution "하:2,중하:4,중:7,중상:5,상:2"
```

문항 형식과 제목도 지정할 수 있습니다.

```bash
python main.py "input/역삼중.hwp" \
  --distribution "하:3,중하:5,중:8,중상:3,상:1" \
  --types "객관식,주관식,서술형" \
  --title "중2 일차부등식 단원평가" \
  --academy "역삼수학학원"
```

## 출력 파일

`workspace/outputs/실행시각/`에 다음 파일이 생깁니다.

- `학생용_문제지.docx`
- `교사용_정답해설.docx`
- `전체_생성결과.json`

문제은행은 `workspace/problem_bank.db`에 누적됩니다.

## 중요 제한사항

- AI 생성 문항은 자동 검사를 통과하더라도 실제 배포 전 교사가 최종 검토해야 합니다.
- 그림 의존성이 큰 도형·그래프 문제는 원본 그림을 완전하게 재구성하지 못할 수 있습니다.
- 제공받은 기출문제는 내부 참고용으로 사용하고, 원문을 그대로 재배포하지 마세요.
- OpenAI API 사용량에 따라 별도 비용이 발생합니다.
